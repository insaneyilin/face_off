# Face Off

A "Face Changing" demo using Python and OpenCV.

## Try Demo

```
# from a video file
python test_video_capture_from_file.py

# from video device 0 (e.g. the camera of your laptap)
python test_video_capture.py
```

## Dependencies

- Python 2.7x
- OpenCV
- NumPy

## References

The video processing framework refers to the book [OpenCV Computer Vision with Python](https://nummist.com/opencv/#book.3923) by [Joseph Howse](http://nummist.com/)
